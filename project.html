Farm Management System

This project is a Farm Management System developed using basic C programming. It allows farmers to efficiently manage daily agricultural activities such as crop records, livestock details, and expense tracking. Designed with a simple console interface, the system aims to streamline farm operations and support better decision-making for small-scale farmers.

Pet Care And Adoption System

This project is a Pet Care and Adoption System designed to help manage pet information, facilitate adoptions, and provide basic care guidelines. The system streamlines the process for pet shelters and individuals looking to adopt, offering features such as pet listings, adopter details, and care tips. It aims to promote responsible pet ownership and improve the adoption experience.

Temperature-Based Fan

This project is a Temperature-Based Fan System built using Arduino. It automatically controls the fan speed based on the surrounding temperature, using a temperature sensor (like LM35 or DHT11) to monitor heat levels. The system helps in energy efficiency and provides a hands-free cooling solution, making it ideal for smart home or automation applications.
